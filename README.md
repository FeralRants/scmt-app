# Service Cloud Migration Tool - Package

Migrate your data from your Desk.com org to your Service Cloud org.

## Supported Data
* Users
* Queues (Groups & Memberships)
* Articles (primary language only)
* Accounts (Companies)
* Contacts (Customers)
* Cases & Attachments
* Custom Fields and Labels
